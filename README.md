# Uday Pratap Yadav — Rank 5, 70th BPSC

A tribute website celebrating the achievement of Uday Pratap Yadav, who secured All India Rank 5 in the 70th BPSC CCE 2026 and was selected as SDM in the Bihar Administrative Service.

## Live

https://neural-arun.github.io/uday_bpsc/

## Built With

- HTML5
- CSS3 (custom design system, no frameworks)
- Vanilla JavaScript

## Features

- Dark midnight blue + gold theme
- Responsive (mobile-first)
- Scroll-triggered animations
- Photo gallery with grid layout
- YouTube video embeds
- Local celebration video
- Gold particle sparkle effect
- Scroll progress bar

## Sections

- Hero — rank, name, portrait
- Journey — timeline with colored milestone cards
- About — background, interests, stats
- Results — rank cards, topper table, press mentions
- Mock Interview — preparation details
- Gallery — photo grid
- Family — photo strip
- Media — video embeds
- Advice for Aspirants
